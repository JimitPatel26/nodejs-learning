# 01 - Node.js Introduction

This folder contains the basic introduction examples for **Node.js**.

## Files

### 1. 01-intro.js

A simple program that demonstrates basic JavaScript logic inside Node.js.

It checks whether a number is small or large and prints a message.

Example output:

```
Large Number
My First Node app !!
```

Run the file:

```
node 01-intro.js
```

---

### 2. app.js

This file demonstrates **Node.js global variables** and the `setInterval()` function.

Globals used:

* `__dirname` → Current directory path
* `__filename` → Current file name
* `require` → Import modules
* `module` → Info about current module
* `process` → Environment information

Example output:

```
D:\nodejs-learning\01-intro
D:\nodejs-learning\01-intro\app.js
Hello World
Hello World
Hello World
...
```

Run the file:

```
node app.js
```
